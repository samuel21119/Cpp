Open problems file
===================================
### To open pdf files, "cd" to this directory (UVa/), and type :
        $ ./macpdf "Problem number"
### to open pdf file (for macOS only)
### or :
        $ ./linuxpdf "Problem number"
### for linux GUI interface only (Okular needed)
